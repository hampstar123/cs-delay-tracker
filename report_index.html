📋 How to load data

Because the data is stored securely in Microsoft Forms on KP servers, you need to paste it in manually each time you want to view the report. Here's how:

  1. Go to forms.microsoft.com and open the Scheduled Delays form
  2. Click the Responses tab → click Open in Excel
  3. Select all the data rows (not the header) → Copy → Paste into the box below
  4. Repeat for Unscheduled Delays and Cerclage Delays
  5. Click Load Data
